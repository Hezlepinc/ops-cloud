# Legacy Theme (Not Deployed)

This directory contains an older, full theme layout kept for reference.

Current deployment uses a single child theme `hello-child` assembled at build time, plus brand overlays:

- Base runtime theme: `infra/wordpress/themes/hello-child/`
- Brand overlay (use this instead of editing here): `infra/wordpress/brands/hezlep-inc/theme/`
- Brand tokens: `infra/wordpress/brands/hezlep-inc/assets/`

Do not edit this folder for live sites. Move any needed templates into the brand overlay.
